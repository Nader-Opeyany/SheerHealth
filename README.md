# Sheer Health NHS Medicines Scraper

This project scrapes, with a slight delay for politeness, the entire [NHS Medicines A-Z](https://www.nhs.uk/medicines/) list, capturing all medicine names, links, and relevant details for each medication.

## Requirements

- **Python 3** (recommended: 3.7+)
- The following Python packages:
  - `requests`
  - `beautifulsoup4`

## Installation

To install the required dependencies, run the following command in your terminal after downloading/cloning the repository:

```bash
pip install requests beautifulsoup4
